QuantaLang

A compiled language with a self-hosting compiler, 62 native programs, and a SQL database engine. Built from scratch in Rust.

# compile and run
$ quantac hello.quanta && ./hello
Hello, QuantaLang!

# the self-hosted compiler compiles itself
$ qcodegen codegen.quanta > codegen.c
2,490 lines .quanta → 3,600 lines C

# a real database engine, written in quantalang
$ qdb app.db "SELECT name, dept FROM employees WHERE salary > 100000 ORDER BY name"
name | dept
-------+-----
Harper | eng
Zain | eng

What it does

Compiler

Rust-based, 80K lines. Lexer, parser, type checker (Hindley-Milner), SSA-based MIR, 8 backends. 507 tests, zero warnings.

Self-Hosting

2,490 lines of QuantaLang that compile QuantaLang to C. Structs, closures, impl methods, Vec, include!(). Generates valid C for all 62 programs.

Database Engine

SQL with JOIN, GROUP BY, HAVING, DISTINCT, indexes, WAL crash recovery, transactions. 4,600 lines.

62 Programs

Near-complete coreutils: grep, sed, awk, sort, diff, patch, find, jq, calc, base64, markdown→HTML, and more.
