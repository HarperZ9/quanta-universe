## 951. Parallel Prefix Sum
**Novelty Class:** Significant Improvement

Work-efficient scan algorithm.

```hlsl
groupshared uint shared_data[256];

[numthreads(128, 1, 1)]
void PrefixSum(uint tid : SV_GroupThreadID, uint gid : SV_GroupID) {
    uint ai = tid;
    uint bi = tid + 128;
    
    // Load
    shared_data[ai] = g_Input[gid * 256 + ai];
    shared_data[bi] = g_Input[gid * 256 + bi];
    
    // Upsweep (reduce)
    uint offset = 1;
    for (uint d = 128; d > 0; d >>= 1) {
        GroupMemoryBarrierWithGroupSync();
        if (tid < d) {
            uint ai = offset * (2 * tid + 1) - 1;
            uint bi = offset * (2 * tid + 2) - 1;
            shared_data[bi] += shared_data[ai];
        }
        offset *= 2;
    }
    
    // Downsweep
    if (tid == 0) shared_data[255] = 0;
    for (uint d = 1; d < 256; d *= 2) {
        offset >>= 1;
        GroupMemoryBarrierWithGroupSync();
        if (tid < d) {
            uint ai = offset * (2 * tid + 1) - 1;
            uint bi = offset * (2 * tid + 2) - 1;
            uint t = shared_data[ai];
            shared_data[ai] = shared_data[bi];
            shared_data[bi] += t;
        }
    }
    
    GroupMemoryBarrierWithGroupSync();
    g_Output[gid * 256 + ai] = shared_data[ai];
    g_Output[gid * 256 + bi] = shared_data[bi];
}
```
